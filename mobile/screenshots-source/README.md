# App Store screenshot source

Add your **raw screenshots** (PNG) here. Any size is fine; the script resizes to:

- **iPhone 6.7"**: 1242×2688 (portrait)
- **iPhone 6.7" landscape**: 2688×1242

Run from repo root:

```bash
npm run generate-app-store-screenshots
```

Output goes to `mobile/app-store-screenshots/`. Use those for App Store Connect.
