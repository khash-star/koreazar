# Workflows

Task playbooks for Cursor. Pick **one** that best matches the request.

| Playbook | When |
|----------|------|
| `new-feature.md` | New user-facing capability |
| `bugfix.md` | Regression or broken behavior |
| `firebase-change.md` | Firestore, Auth, Storage, indexes, rules |
| `mobile-change.md` | Expo app only (`mobile/`) |

Always complete `../senior-developer/TASK_WORKFLOW.md` intake and handoff steps.
