# Templates

Copy structure into Cursor replies, PRs, or `decisions/` when filling ADRs.

| Template | Use |
|----------|-----|
| `change-report.md` | Required after code changes |
| `pr-description.md` | Pull request body |
| `adr.md` | Architecture decision record |
| `memory-update-report.md` | After updating project-memory files |
| `test-report.md` | After QA / regression testing |
