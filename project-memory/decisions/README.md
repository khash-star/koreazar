# Architecture Decisions

Human-maintained ADRs for significant choices. AI may **propose** using `../templates/adr.md` but should not invent production decisions without review.

## Naming

`ADR-0001-short-title.md`

## When to add an ADR

- New data model or collection layout  
- Auth or admin model change  
- Mobile vs web split strategy  
- PWA/TWA vs native store approach  

## Decision log (chronological)

| File | Use |
|------|-----|
| `DECISION_LOG.md` | Major decisions with date, reason, risks, rollback |
